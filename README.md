# STAFF

«STAFF» - веб-приложение для хранения и управления информацией о сотрудниках организации.

Проект развёрнут на хостинге Railway.

Ссылка на сайт: https://employee-catalog-production.up.railway.app/

## Доступ к системе

Самостоятельная регистрация пользователей отсутствует.

Учётные записи создаются администратором системы. Для входа пользователю необходимо получить:

* Email;
* Пароль;

После авторизации пользователь получает доступ в соответствии со своей ролью.

## Основной функционал

### Каталог сотрудников

Для каждого сотрудника отображаются:

* ФИО;
* Отдел;
* Должность;

При нажатии на кнопку **«Подробнее»** открывается карточка сотрудника со следующими данными:

* Описание;
* Телефон;
* Email;
* Дата найма;
* Отдел;
* Должность;

Поддерживаются:

* Поиск по имени, отделу, должности;
* Фильтрация по отделам;
* Фильтрация по должностям;

### Административная панель

Администратор может:

* Добавлять новых сотрудников;
* Редактировать данные сотрудников;
* Удалять сотрудников;
* Создавать новые отделы;
* Создавать новые должности;
* Экспортировать сотрудников в формат CSV;

Каждая должность связана с определённым отделом.

### Панель аккаунтов

Содержит список всех пользователей, имеющих доступ к системе.

Для каждого аккаунта отображаются:

* Email;
* Пароль (Скрыт);
* Уровень доступа;

Администратор может:

* Создавать аккаунты;
* Изменять данные аккаунтов;
* Удалять аккаунты.

Всего имеется 3 уровня доступа:

* Сотрудник — имеет доступ к каталогу сотрудников;
* HR — имеет доступ к административной панели;
* Администратор — обладает полным доступом к системе, включая панель аккаунтов;

Каждый аккаунт обязательно связан с уже существующим сотрудником.
